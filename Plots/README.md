# Plots

This directory contains all intermediate and final data used for figure generation in this study.

## Folder structure

- **figures/**  
  Final figures exported from the analysis scripts (PDF and PNG formats).

- **tables/**  
  Data tables used to generate figures and statistical summaries.

- **stats/**  
  Statistical test outputs and summary statistics used in the analyses.

- **processed/**  
  Processed datasets prepared for visualization and downstream analyses.

- **layers/**  
  Intermediate data layers used to build composite plots and visualizations.

- **trees/**  
  Phylogenetic tree files used in evolutionary analyses.

- **consenTrait/**  
  Input and output files for ConsenTrait phylogenetic signal analysis.
